#### 题目：17.电话号码的字母组合

时间：2019.09.21

##### 题干：

```javascript
给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。

给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。

```

##### 示例：

```
输入："23"
输出：["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"].

尽管上面的答案是按字典序排列的，但是你可以任意选择答案输出的顺序
```

##### 解答：

```javascript
var letterCombinations = function(digits) {
    //空串
    if(!digits){
        return [];
    }
    //字典数据
    var dict= {
        1: [],
        2: ['a','b','c'],
        3: ['d','e','f'],
        4: ['g','h','i'],
        5: ['j','k','l'],
        6: ['m','n','o'],
        7: ['p','q','r','s'],
        8: ['t','u','v'],
        9: ['w','x','y','z']
    };

    var arr = []; //缓存数据
    var func = function(cacheArr, targetChar) {
        let targetCharArr = dict[targetChar];
        let targetArr = [];
        targetCharArr.forEach((target) => {            
            if (cacheArr.length) {
                cacheArr.forEach(dict => {
                    targetArr.push(dict + target)
                })
            }else {
                targetArr.push(target);
            }
        })     
        arr = targetArr;        
    }
    //因为循环的的次数不定，所以采用递归的形式
    let i = 0;
    while(digits[i]) {
        func(arr,digits[i])
        i++;
    }
    return arr;
};
```
